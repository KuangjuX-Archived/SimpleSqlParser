# SimpleSqlParser
一个简单的 sql 词法分析器和语法分析器

## Usage
运行词法分析器与语法分析器：  

```shell
make run
```  
  
运行 NFA 到 NFA 的确定化与 DFA 的最小化：

```shell
make dfa
```
